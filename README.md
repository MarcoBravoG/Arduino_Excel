Comunicación serial Arduino – Excel


El siguiente código adquiere una señal analógica mediante el puerto analógico A0 de Arduino, dicha señal se gráfica utilizando Excel.
Materiales:
-	Arduino nano
-	Potenciómetro de 2Kohm
-	Protoboard
-	Cables machos
-	Cable miniusb
-	Software Arduino
-	Software Excel
